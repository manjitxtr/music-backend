# Music Subscription Backend (Node.js + Express)

## Project Setup (Run Locally)

Follow the steps below to run the backend server on your local machine.

---

## 1. Prerequisites

Make sure the following are installed:

- Node.js (v18 or above recommended)
- npm (comes with Node.js)
- Git

---

## 2. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/music-subscription-backend.git
cd music-subscription-backend
```

## 3. Install Dependencies
```bash
npm install
```

## 4. Running the Backend
```bash
node server.js
```
